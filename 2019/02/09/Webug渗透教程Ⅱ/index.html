Webug渗透教程Ⅱ

第二题：图片隐写，Winhex。第三题：扫描目录， 御剑。第四题：爆破密码， Burp suite

从图片中你能找到什么？

  1. 图片保存本地，用Winhex打开。

  2. 改后缀zip，打开文件。

你看到了什么？

  1. 查看源码

  2. 用御剑扫一下

  3. 打开网址

  4. MD5加密字符“test”

  5. 访问[http://192.168.88.141/pentest/test/san/4621d373cade4e83/]

告诉你了 FLANG 是五位数

  1. 这里忘记密码和立即注册都没用，被阉割了。只能Burp来爆破密码了

  2. 在拦截的请求页面，右键点击 send to intruder

  3. 在 Positions 里打了$$的就是变量，我们先clean掉，再自行add$变量。

    如果只有一个变量，Attack type 选择 sniper、两个以上则选 cluster bomb。

  4. 转到Payloads，Payload set 选择爆破的变量，Load 字典。右上角点击 Start attack。

  5. 对爆破结果的长度做个排序，得到 用户名 密码：admin/admin123。

  6. 登陆，成功
