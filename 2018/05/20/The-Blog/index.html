博客的创建历程

踩过的坑

  1. 安装node.js/Hexo的时候，一定要在管理员权限下进行，无论是Liunx/Windows。

  2. npm是node.js自带的，安装完node.js后，直接在cmd中执行npm install -g hexo

  3. Hexo _config.yml 配置文件中，Git 的配置应该这样

  4. Hexo d 推到Github的时候，有时候会因为网速问题，停顿好久（卡着不动），这个不要心急，去看会直播什么的，等它上传完后就好啦。

  5. 每次修改完Hexo配置或文件后，需要先 ‘hexo clean’ 再 ‘hexo g’ 最后再运行 ‘hexo s’。

  6. 关于about页面，在命令行中执行’hexo new page “about”‘，’source’文件夹下就会出现一个’about’文件夹，自行更改’index.md’的内容。
